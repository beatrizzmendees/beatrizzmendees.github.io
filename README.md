Projeto de um currículo online desenvolvido com HTML, CSS e JavaScript, com foco principal em praticar conceitos de Git e GitHub.

Objetivos:
Treinar versionamento com Git
Aprender fluxo de trabalho no GitHub
Praticar commits, branches e organização de repositórios
Publicar projetos utilizando GitHub Pages

Tecnologias Utilizadas: HTML5, CSS3, JavaScript, Git e GitHub.
